Boundaries

Boundaries

Audit results are useful only when read with the supplied model, configured bound, assumptions, and exclusions.

What the Result Means

UNSAFE

A bad path exists in the supplied model within the configured bound.

SAFE_WITHIN_BOUND

No bad path was found within the supplied model and configured bound.

What Remains Outside Scope

  • It does not cover behavior outside the supplied model.
  • It does not cover all real-world system behavior.
  • It does not replace security review.
  • It does not model probabilistic LLM/tool behavior.
  • It does not perform full IAM/cloud analysis.

Technical Context

This demo is developed alongside broader formal research, but the public site presents only the bounded workflow audit artifact. Evaluate it on the scoped model, bound, verifier result, trace, remediation, and retest. Formal context is separated into the technical appendix.
