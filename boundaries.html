Boundaries

Boundaries

Track B results are useful only when read with the supplied model, configured bound, assumptions, and exclusions.

What the Result Means

UNSAFE

A bad path exists in the supplied model within the configured bound.

SAFE_WITHIN_BOUND

No bad path was found within the supplied model and configured bound.

What the Result Does Not Mean

  • It does not cover behavior outside the supplied model.
  • It does not cover all real-world system behavior.
  • It does not replace security review.
  • It does not model probabilistic LLM/tool behavior.
  • It does not perform full IAM/cloud analysis.

Formal Work Boundary

Track B is developed alongside broader CRSL/RSVT formal research, but this site presents only the bounded workflow audit demo. The demo should be evaluated on its scoped artifact: model, bound, verifier result, trace, remediation, and retest.
